# Bookly

![Logo](img/logo.png)

Bookly é um programa que permite a criação de clubes do livro digitais onde os usuários podem criar encontros e votações de forma simples e eficaz.

## Principais funcionalidades
Confira abaixo algumas funcionalidades do sistema:

1. **Criação de clubes do livro:** qualquer usuário poderá criar o seu próprio clube do livro;
2. **Registro de livros:** os administradores de clubes do livro podem adicionar livros novos no banco de dados;
3. **Criação de votações:** os administradores podem criar votações para definir o livro e a data limite para serem lidos;
4. **Criação de encontros:** os administradores podem definir encontros para discussão entre os participantes.
